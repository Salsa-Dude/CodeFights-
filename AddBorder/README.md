
<h2>Add Border</h2>
<p>Given a rectangular matrix of characters, add a border of asterisks(*) to it.</p>

<strong>Example</strong>
For <br>
picture = ["abc",
           "ded"]
<br>
the output should be <br>
addBorder(picture) = ["*****",
                      "*abc*",
                      "*ded*",
                      "*****"]
